# 🧠 Mental Health Analysis System

## 📌 Project Overview

This project aims to analyze user text and predict:

* Emotion
* Sentiment
* Stress/Risk level

The system combines NLP techniques and machine learning models.

---

## ⚙️ Tech Stack

* Python
* Flask (Backend API)
* Transformers (BERT - planned)
* Scikit-learn
* Pandas
