🌸 MindBloom Frontend

A clean and intuitive frontend interface for MindBloom, a wellness and journaling web application designed to help users track their mood, reflect on their thoughts, and build healthier habits.

Features
 Daily mood check-ins
 Personal journaling system
 Mood trends visualization
 Calendar for tracking events and deadlines
 Weekly reflection questionnaire
 Clean, aesthetic, and user-friendly UI
 
Tech Stack
React (Vite)
JavaScript (ES6+)
HTML5 & CSS3
Inline styling (custom UI components)

Getting Started
1. Clone the repository
git clone <repo-link>
cd Major-Project/frontend
2. Install dependencies
npm install
3. Run the development server
npm run dev

Then open the app in your browser at:

http://localhost:5173
